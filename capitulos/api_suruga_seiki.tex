\chapter{Documentacion de la API Suruga Seiki}
\label{ap:api-suruga-seiki}

Es necesario cargar el ensamblado (assembly), una unidad de código previamente compilada que permite establecer la comunicación con la plataforma Suruga Seiki. Este ensamblado está compuesto por un conjunto de clases que definen las propiedades (atributos) y los métodos (acciones o funcionalidades) disponibles en la plataforma.

En programación orientada a objetos, los métodos representan las acciones que una clase puede ejecutar, mientras que las propiedades describen las características asociadas a sus objetos. Un objeto es una instancia concreta de una clase; utilizando una analogía, la clase puede entenderse como un plano de construcción y el objeto como el resultado material obtenido a partir de ese plano.
